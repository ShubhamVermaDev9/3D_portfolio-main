## 🔮 &nbsp; Overview

> **A fully immersive, interactive 3D portfolio** built with cutting-edge web technologies — featuring real-time 3D models, fluid animations, and a sleek dark UI that pushes the limits of what's possible in a browser.

This portfolio is not just a website — it's an **experience**. Every scroll, hover, and click is crafted with intention, combining the power of **Three.js** for 3D rendering, **GSAP** for timeline animations, and **Framer Motion** for buttery-smooth UI transitions — all wrapped in a blazing-fast **React** app styled with **Tailwind CSS**.

## ✨ &nbsp; Features

| 🔮 Feature | 📌 Description |
|:---|:---|
| **3D Hero Scene** | Interactive Three.js canvas with real-time 3D models & lighting |
| **GSAP Animations** | Scroll-triggered timeline animations throughout the page |
| **Framer Motion** | Page transitions & micro-interactions on every element |
| **About Me** | Personal intro with animated skill display |
| **Projects Showcase** | Featured work with live links & tech stack tags |
| **Contact Section** | Functional contact form with animated UI |
| **Fully Responsive** | Optimized for desktop, tablet & mobile |
| **Vercel Deployed** | Lightning-fast CDN delivery via Vercel |

## 🚀 &nbsp; Getting Started

### Prerequisites

Make sure you have the following installed:

```bash
node  >= 18.x
npm   >= 9.x
```

### Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/ShubhamVermaDev9/shubham-3d-portfolio.git

# 2. Navigate into the project
cd shubham-3d-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

> 🌐 Open [http://localhost:5173](http://localhost:5173) in your browser to view it locally.

### Build for Production

```bash
# Create an optimized production build
npm run build

# Preview the production build locally
npm run preview
```

## 🌍 &nbsp; Deployment

This project is deployed on **Vercel** with zero-config CI/CD.

Every push to `main` automatically triggers a new deployment.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/)

```bash
# Or deploy manually via Vercel CLI
npm i -g vercel
vercel --prod
```
